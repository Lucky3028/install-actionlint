# install-actionlint

An action to install [rhysd/actionlint](https://github.com/rhysd/actionlint).

## Usage

```yaml
  - uses: Lucky3028/install-actionlint@v1
    with:
      version: 0.2.0
```

### Inputs

|name|description|required|
|---|---|:---:|
|version|The version of actionlint to install. (ref. 1.1.0) If not provided, use the latest.|â¬œ|
